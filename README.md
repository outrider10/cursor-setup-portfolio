# Cursor Setup Portfolio

## Tools Installed

- Cursor IDE

- Claude Code Extension

- Codex Extension

- Git

- GitHub

## Steps Completed

1. Installed Cursor IDE.

2. Installed the Claude Code extension.

3. Installed the Codex extension.

4. Created a public GitHub repository.

5. Installed Git.

6. Cloned the repository using Git.

7. Opened the repository in Cursor.

## Issues Encountered

- I was new to Git and GitHub and had never used them before.

- Initially, Git commands were unavailable because Git was not installed.

- After installing Git and restarting Cursor, I was able to clone the repository successfully.

- I learned the difference between Git (version control) and GitHub (online repository hosting).
